# recallai-website

这是一个简单的 React 静态前端示例（RecallAI 项目展示页）。

快速开始：

1. 安装依赖：

```powershell
npm install
```

2. 本地开发：

```powershell
npm run dev
```

3. 构建：

```powershell
npm run build
```

把项目推送到 GitHub 的示例命令（如果你已登录 `gh`）：

```powershell
gh repo create recallai-website --public --source=. --remote=origin --push
```

如果没有 `gh` 可使用：

```powershell
git remote add origin https://github.com/你的用户名/recallai-website.git
git branch -M main
git push -u origin main
```
